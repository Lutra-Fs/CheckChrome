# UpdateTracker

Update tracker for Google Chrome and some other packages.

## Packages

- Google Chrome
    - `extras/googlechrome`
    - `extras/googlechrome-portable`
    - `extras/googlechrome-beta`
    - `extras/googlechrome-dev`
    - `extras/googlechrome-canary`
- Ryzen Controller (`extras/ryzen-controller`)
